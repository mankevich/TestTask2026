# Тестовое задание: Менеджер подписок (Манкевич В.А.)

Реализация интерфейса управления подписками на сайты с динамическим расчетом показателя "FUN YOU GET".

## Стек:
*   **Framework:** Vue 3 (Composition API)
*   **Language:** TypeScript
*   **State Management:** Pinia
*   **Build Tool:** Vite
*   **Styling:** SCSS, адаптивная и семантичная верстка.

## Реализованный функционал:
1.  **Загрузка данных:** Список сайтов и начальные состояния берутся из `subscribe.json`.
2.  **Управление подписками:** Индивидуальные переключатели и кнопка "Unsubscribe from all".
3.  **Динамический расчет:** Прогресс-бар "FUN YOU GET" обновляется мгновенно при изменении подписок.
4.  **Попап:** При массовой отписке отображается окно с расчетом разницы потерянного "фана".
5.  **Логирование:** При нажатии кнопок текущее состояние формируется в JSON-объект и выводится в консоль браузера (согласно ТЗ).

## Как запустить:

1. Установка зависимостей:
   ```bash
   npm install
   ```

2. Запуск в режиме разработки:
   ```bash
   npm run dev
   ```

3. Сборка проекта:
   ```bash
   npm run build
   ```

## Примечания:
*   Библиотеки стилей (Bootstrap и др.) не использовались.
*   Верстка полностью адаптивна.
